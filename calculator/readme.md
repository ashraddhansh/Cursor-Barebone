# Calculator App

This is a simple command-line calculator application written in Python.

## Features

*   **Evaluates mathematical expressions:** The application can evaluate basic mathematical expressions provided as a string.
*   **Command-line interface:** It takes the expression as a command-line argument.
*   **Error handling:** It provides basic error handling for invalid expressions.

## Usage

To use the calculator, run the `main.py` script with your desired expression enclosed in quotes:

```bash
python main.py "3 + 5"
```

## Example

```bash
python main.py "10 / 2 - 1"
```

This will output:

```
10 / 2 - 1 = 4.0
```
